# ATM Simulation Project

This is a Python-based ATM simulation system.  
It allows users to perform basic banking operations like checking balance, depositing money, withdrawing money, and viewing transaction history.

## Features
- User authentication using PIN
- Check account balance
- Deposit money
- Withdraw money
- Transaction history tracking

## How to Run
1. Clone or download the repository
2. Open in VS Code or any Python IDE
3. Run the file:
   python atm.py

## Tech Stack
- Python 3.13.13
